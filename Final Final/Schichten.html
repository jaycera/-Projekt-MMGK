Die Schichten des Internets

"World Wide Web" by Elena Regina is marked with CC PDM 1.0

Cyan: Clear Web/Surface Web
Rot: Deep Web
Magenta: Dark Web
Magenta und Rot bilden dabei eine Einheit, die weiter unterteilt wurde.

Man teilt das Internet in drei Schichten auf: das Clear Web (auch Visible Web oder Surface Web), das Deep Web (auch Hidden Web oder Invisible Web) und das Dark Web oder Darknet.

Clear Web

Das sind die Seiten, die wir ganz normal mit Suchmaschinen wie Google, Bing und Yahoo finden und ohne Anmeldung nutzen können. Wann kommt die Straßenbahn oder wie wird das Wetter morgen? Zwei Klicks und wir haben die Antwort, einfach und bequem und ohne das Clear Web zu verlassen.

Deep Web

"File:Deep-web-1292332 1280-1024x774.jpg" by Gerd Altmann is marked with CC0 1.0

Das Deep Web macht etwa 90% des Internets aus. Hierbei handelt es sich (eigentlich) nicht um illegale Seiten, sondern um Seiten mit beschränktem Zugang. Also ist jede Seite, für die man sich anmelden oder einloggen muss, schon im Deep Web verankert. Seiten im Deep Web sind im Normalfall von Firewalls oder Paywalls (man muss bezahlen um bestimmte Inhalte zu sehen) geschützt, größtenteils handelt es sich um Datenbanken. Schnell ein Bild bei Instogram posten oder die Pizza über Pay Pal bezahlen? Wilkommen im Deep Web.

Dark Web

Das Dark Web ist ein Teil des Deep Webs. Seiten im Dark Web sind nur mit spezieller Software, zum Beispiel dem TOR Browser, erreichbar. Die wichtigsten Eigenschaften des Darknets sind hierbei, dass der Datenverkehr durch Verschlüsselung anonym und die Nutzer selten zurückzuverfolgen sind. Dabei gibt es im Darknet nicht nur illegale Seiten. Facebook zum Beispiel hat auch eine Darknet-Präsenz, so ist es in Ländern, in denen die Social Network Seite verboten ist, trotzdem zugänglich. Nach der gleichen Idee ist auch die Nachrichtenplattform ProPublica ist im Dark Web zu finden. Da das Dark Web fast absolute Anonymität bietet, wirkt es auf Kriminelle natürlcih sehr anziehend. IP-Adressen sind nur schwer zurückzuverfolgen und alles wird verschlüsselt, das bietet eine ideale Plattform für illegale Aktivitäten.
